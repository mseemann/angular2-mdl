List components
Component Description
mdl-list Basic container for any mdl-list component.
mdl-list-item Defines a item in the list. The attribute lines will be used to specify of how many individual lines a list item consist. You can use 1, 2 and 3. 1 is the default value.
mdl-list-item-primary-content Defines the primary content sub-division.
mdl-list-item-secondary-action Defines the Action sub-division. Needs a 2 or 3 line list-item.
mdl-list-item-secondary-content Defines the secondary content sub-division. Needs a 2 or 3 line list-item.
mdl-list-item-secondary-info Defines the information sub-division. Needs a 2 or 3 line list-item.
mdl-list-item-sub-title Defines the sub title in the mdl-list-item-primary-content component.
mdl-list-item-text-body Defines the text-section in the mdl-list-item-primary-content component.
Additional attributes for mdl-icon

These attributes can be used to style mdl-icons in lists

Attribute Description
mdl-list-item-avatar> avatar icon.
mdl-list-item-icon Small icon.
mdl-ripple Add mdl-ripple to the mdl-list-item component to create a ripple effect.
