Card-Components

Component Description
mdl-card Basic components that creates a card
mdl-card-actions Container for actions at the bottom of the card.
mdl-card-media A media container within an mdl-card.
mdl-card-menu A menu for the card. Usually an mdl-button with mdl-button-type icon.
mdl-card-supporting-text Section within an mdl-card. Can be providedmultiple times.
mdl-card-title Section within an mdl-card.

Card-Component Attributes

Attribute Description
mdl-card-border Creates a border around a card section.
mdl-card-expand Adds flex-grow:1 to a card section so that the section expands as horizontally as much as possible.
mdl-card-title-text Specifies a Title of a card. Every h* tag should have the attribute mdl-card-title-text.
